## Deploy to Vercel (≈10 minutes)

1. **Buy the domain** and clear the trademark screen first (see launch checklist below).
2. **Create a GitHub repo** and push this `08_Website/` folder as the repo root:
   ```bash
   cd "08_Website"
   git init && git add . && git commit -m "Sentinel site v1"
   git branch -M main
   git remote add origin https://github.com/<you>/sentinel-site.git
   git push -u origin main
   ```
3. **Import to Vercel:** vercel.com → Add New → Project → import the repo. Framework preset: **Other**. Root directory: `./`. No build command, output dir = root. Deploy.
4. **Add your domain** in Vercel → Project → Settings → Domains, and point your registrar's nameservers/records as Vercel instructs. SSL is automatic.
5. `robots.txt` and `sitemap.xml` are already set to **sentinelassurance.group** — submit the sitemap in Google Search Console after the domain is live.
